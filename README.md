# wtop

> Windows 작업표시줄 위에 상시 떠 있는 가벼운 리소스 모니터

맥 메뉴바처럼 CPU·RAM·디스크·네트워크 상태를 한눈에 보여주고,
시스템이 폭주해 작업관리자가 안 열릴 때 쓸 수 있는 비상용 프로세스 패널을 제공합니다.

## 다운로드

**[최신 버전 받기 →](https://github.com/thebschoi-corp/wtop-releases/releases/latest)**

| 파일 | 용도 |
|---|---|
| `wtop-setup-x.x.x.exe` | 설치 프로그램 (권장 — 관리자 권한 불필요) |
| `wtop-portable-x.x.x.zip` | 무설치 버전 (압축 풀고 wtop.exe 실행) |

> **SmartScreen 안내**: 서명되지 않은 파일이라 처음 실행 시 "Windows의 PC 보호" 경고가 뜰 수 있습니다.
> `추가 정보 → 실행`을 누르면 됩니다.

## 주요 기능

- 작업표시줄 위 상시 표시 (다른 창에 가려지지 않음)
- CPU / RAM / 디스크 / 네트워크 / 배터리 실시간 표시
- 위젯 우클릭 → 프로세스 패널 (검색·정렬·프로세스 종료)
- 임계치 경고, 전체화면 자동 숨김, 멀티모니터 지원
- 새 버전이 나오면 자동 알림

## 사용법 요약

| 동작 | 기능 |
|---|---|
| 위젯 드래그 | 위치 이동 (자동 저장) |
| 위젯 우클릭 | 프로세스 패널 열기/닫기 |
| 트레이 아이콘 우클릭 | 설정 메뉴 |
